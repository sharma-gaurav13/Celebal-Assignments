DATA EXPLORATION AND CLEANING ASSIGNMENT:

This repository contains my submission for the data cleaning assignment. The project focuses on taking a raw, unformatted shopping dataset, handling structured missing information, formatting mixed data types, and cleaning the records for proper numerical calculation.

WHAT THIS CODE DOES:

1. Initial Exploration: Reads the raw Combined_dataset.csv file to check the overall dimensions, columns, and data types.

2. Handling Missing Values: Fills missing values in crucial columns like discount, rating, and ratings_count with logical defaults (0 or Unknown).

3. Currency Data Formatting: Fixes the string conversion lockups in columns like final_price by stripping away text elements (the rupees symbol and commas) and converting them into numerical floats.

4. Data Deduplication: Checks for and removes duplicate rows to ensure clean analysis records.

5. Feature Calculation: Creates a new engineered column total_amount based on clean mathematical calculations (final_price multiplied by quantity).

6. Data Export: Saves the final, completely clean dataframe into cleaned_shopping_dataset.csv.


FILES INCLUDED:

- analysis.ipynb: The primary Jupyter Notebook containing the full Python pipeline script.
- cleaned_shopping_dataset.csv: The finalized clean dataset generated by running the code.


ENVIRONMENT:

- Python 3.14 
- Pandas and NumPy libraries
